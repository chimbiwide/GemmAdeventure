# GemmAdventure

## An AI powered dungeon crawler using our custom `Gemaa3NPC-it` model

---

#### How to play

Run `game.py` to start the game, our evil magician will greet you, good luck!

#### Some commands

- `/exit`: exit the game without saving
- `/save`: save the game in a json format
- 'resume': resume the game from the `history.json` file

#### Where to get the model?

Visit our [Huggingface page](https://huggingface.co/npcLM)

#### Dependencies

the `lmstudio` python sdk is required.
`uv` is also strongly recommended
